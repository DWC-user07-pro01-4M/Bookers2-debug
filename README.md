## インストール方法
* 以下のコマンドを上から順に実行してください。
```
git clone git@github.com:DWC-user07-pro01-4M/Bookers2-debug.git
```

```
cd bookers2-debug
```

```
rails db:migrate
```

```
yarn install
```

```
bundle install
```
* rails sでサーバーを立ち上げ、ブラウザで閲覧できれば成功です。
